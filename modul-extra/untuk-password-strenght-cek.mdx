### **Fungsi JavaScript**
```javascript
function isValidPassword(password) {
    const minLength = 8;
    const numberRegex = /\d/; // Cek minimal satu angka
    const symbolRegex = /[!@#$%^&*\-+]/; // Cek minimal satu simbol dari daftar

    if (
        password.length >= minLength &&
        numberRegex.test(password) &&
        symbolRegex.test(password)
    ) {
        return true;
    }
    return false;
}

// Contoh Penggunaan
console.log(isValidPassword("Test123!")); // âœ… true
console.log(isValidPassword("12345678")); // âŒ false (tidak ada simbol)
console.log(isValidPassword("password!")); // âŒ false (tidak ada angka)
console.log(isValidPassword("Pass1!")); // âŒ false (kurang dari 8 karakter)
console.log(isValidPassword("StrongPass1@")); // âœ… true
```

### **Penjelasan**:
1. **Cek panjang password** (`password.length >= 8`)
2. **Cek apakah mengandung angka** (`/\d/` â†’ minimal satu angka)
3. **Cek apakah mengandung simbol** (`/[!@#$%^&*\-+]/` â†’ minimal satu simbol dari daftar yang diizinkan)
4. Jika semua kondisi terpenuhi, return `true`, jika tidak return `false`.
