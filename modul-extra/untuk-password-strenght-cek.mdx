Berikut adalah fungsi JavaScript untuk memeriksa apakah password memenuhi kriteria berikut:  
✅ **Minimal 8 karakter**  
✅ **Mengandung setidaknya satu angka** (`0-9`)  
✅ **Mengandung setidaknya satu simbol** dari daftar `! @ # $ % ^ & * - +`  

### **Fungsi JavaScript**
```javascript
function isValidPassword(password) {
    const minLength = 8;
    const numberRegex = /\d/; // Cek minimal satu angka
    const symbolRegex = /[!@#$%^&*\-+]/; // Cek minimal satu simbol dari daftar

    if (
        password.length >= minLength &&
        numberRegex.test(password) &&
        symbolRegex.test(password)
    ) {
        return true;
    }
    return false;
}

// Contoh Penggunaan
console.log(isValidPassword("Test123!")); // ✅ true
console.log(isValidPassword("12345678")); // ❌ false (tidak ada simbol)
console.log(isValidPassword("password!")); // ❌ false (tidak ada angka)
console.log(isValidPassword("Pass1!")); // ❌ false (kurang dari 8 karakter)
console.log(isValidPassword("StrongPass1@")); // ✅ true
```

### **Penjelasan**:
1. **Cek panjang password** (`password.length >= 8`)
2. **Cek apakah mengandung angka** (`/\d/` → minimal satu angka)
3. **Cek apakah mengandung simbol** (`/[!@#$%^&*\-+]/` → minimal satu simbol dari daftar yang diizinkan)
4. Jika semua kondisi terpenuhi, return `true`, jika tidak return `false`.

🛠 **Fungsi ini bisa langsung digunakan dalam validasi form atau backend security checks.** 🚀  
Jika ingin kriteria tambahan seperti huruf besar & kecil, bisa dikembangkan lebih lanjut! 💡
